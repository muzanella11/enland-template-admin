Simple Alerts

Success! This is success message. 
<div class="alert alert-success">
    <strong>Success!</strong> This is success message.
</div>
Info! This is info message displaying some information. 
<div class="alert alert-info">
    <strong>Info!</strong> This is info message displaying some information.
</div>
Warning! This indicates a warning that need attention. 
<div class="alert alert-warning">
    <strong>Warning!</strong> This indicates a warning that need attention.
</div>
Danger! This show danger ie. some negative action. 
<div class="alert alert-danger">
    <strong>Danger!</strong> This show danger ie. some negative action.
</div>

Dismissible Alerts

Success! This is success message. 
<div class="alert alert-success">
    <button type="button" class="close" data-dismiss="alert"><span>&times;</span></button>
    <strong>Success!</strong> This is success message.
</div>
Info! This is info message displaying some information. 
<div class="alert alert-info">
    <button type="button" class="close" data-dismiss="alert"><span aria-hidden="true">&times;</span></button>
    <strong>Info!</strong> This is info message displaying some information.
</div>
Warning! This indicates a warning that need attention. 
<div class="alert alert-warning">
    <button type="button" class="close" data-dismiss="alert"><span aria-hidden="true">&times;</span></button>
    <strong>Warning!</strong> This indicates a warning that need attention.
</div>
Danger! This show danger ie. some negative action. 
<div class="alert alert-danger fade in">
    <button type="button" class="close" data-dismiss="alert"><span aria-hidden="true">&times;</span></button>
    <strong>Danger!</strong> This show danger ie. some negative action.
</div>
